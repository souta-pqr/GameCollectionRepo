# ゲームコレクション

このリポジトリは、以下の4つのブラウザベースのゲームを含んでいます。

1. 数当てゲーム
2. ジャンケンゲーム
3. タイピングゲーム
4. ブロック崩しゲーム

## 数当てゲーム

1～100までの値を予想するゲームです。

## ジャンケンゲーム

コンピュータとジャンケンで勝負するゲームです。

## タイピングゲーム

表示された文字をタイピングしてスコアを稼ぐゲームです。

## ブロック崩しゲーム

キーボード（← →)を利用して、ブロックを崩しゲームです。
